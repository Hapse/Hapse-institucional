# 🌱 Hapse Consultoria Agroalimentar  

Este repositório contém o código-fonte da página institucional da **Hapse Consultoria Agroalimentar**.  

O objetivo do projeto é apresentar os serviços, diferenciais e informações institucionais da empresa de forma moderna, responsiva e otimizada para a web.

---

## 📦 Instalação

Clone o repositório:

```bash
git clone https://github.com/usuario/repositorio.git
cd repositorio
```

Instale as dependências:

```bash
yarn
```

ou

```bash
yarn install
```

---

## 💻 Executando o projeto em ambiente de desenvolvimento

Para iniciar o servidor local:

```bash
yarn dev
```

Após iniciar, acesse no navegador:

```
http://localhost:3000
```

---

## 🏗️ Gerando build para produção

Para gerar a versão otimizada para produção:

```bash
yarn build
```

Caso seja necessário visualizar a build:

```bash
yarn start
```

## 🤝 Contribuição

1. Crie uma nova branch:
   ```bash
   git checkout -b minha-feature
   ```
2. Faça suas alterações
3. Commit:
   ```bash
   git commit -m "feat: descrição da alteração"
   ```
4. Envie para o repositório:
   ```bash
   git push origin minha-feature
   ```
5. Abra um Pull Request
